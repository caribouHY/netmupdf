# Changelog

## [0.2.0] - 2026-06-11

### Added

- CLIで変換済みセクション数、割合、処理中セクション名を表示する進捗表示
- CLIの`--jobs`オプションと`convert_pdf()`の`jobs`引数による並列数指定
- CLIの`--legacy`オプションと`convert_pdf()`の`legacy`引数による旧Markdown抽出モード

### Changed

- PDFセクション抽出をCPU数に応じて自動並列化し、ワーカー障害時は直列処理へフォールバック
- 速度を優先する旧Markdown抽出モードでは、現行モードと出力構造やヘッダー・フッター処理が異なる場合がある
